# 스터디 컨텍스트 — 인프피솔루션

## 정체
팀스파르타 사내 스터디. 우리 모두 팀스파르타 직원이야.

## 이름 유래
리더 **최웅준**과 부리더 **양기철**이 둘 다 INFP라서 "인프피솔루션" — 살짝 농담 섞인 작명.

## 역사
- 처음엔 2명으로 시작한 비공식 모임
- 지금은 16명까지 커진 사내 컬트(?) 같은 분위기
- 최근 창조주가 여러 발표를 하면서 "에이전트 만들기"에 대한 관심이 폭발 → 정식 스터디 주제로

## 일정
- 매주 **수요일 12:30-14:00**
- 장소: 사내 **운동장1**
- 식사 같이 하면서 진행 (점심 시간 활용)

## 현재
- 지금은 **{{weekLabel}}** 진행 중

## 주차 히스토리
**1주차 (5/20) = OT 수준**:
- 하필 그날 Railway 서버 장애가 터져서 급하게 창조주 서버로 임시 이전 배포했어
- 좀 우당탕탕이었음 ㅋㅋ
- 결국 1주차엔 사람들이 **실제 슬랙 연결까지는 못 했음**
- 그래서 1주차는 사실상 OT, 본격 시작은 2주차부터.

**2주차 (오늘) = 본격 시작**:
- 목표: **빙고 3줄 이상 완성**
- 핵심 작업: 슬랙에서 본인이 태그(멘션)되면 텔레그램으로 알림 받기 + 답변 제안까지
- 자유도 높음 — 본인이 만들고 싶은 형태로 자유롭게 작업 가능

## 창조주가 만든 예시
- 멘션 받으면 → **3가지 스타일의 답변을 제안**해주는 에이전트
- 텔레그램 메시지에 버튼 3개 → 누르면 **실제로 슬랙에 답장 즉시 전송**

## 학습자가 만들 수 있는 다른 예시
- 받은 메시지에 자동으로 이모지(👀, ✅ 등) 다는 에이전트
- 분류해서 우선순위별 다른 알림 톤
- 본인 일정과 비교해서 "지금 답장 가능/불가능" 판단해주는 에이전트
- 슬랙 검색해서 관련 과거 대화 같이 보여주기
- 자유 — 에이전트는 어떻게든 만들 수 있음

## 학습자 프로필
- 16명, 대부분 **운영·PM·디자인 직군** (비개발자 다수)
- 터미널/git 처음인 사람이 많음
- 학습 편차가 큼 → 일방적 강의 X → 빙고 형태 개인 페이스 진행
- "내가 뭔가 만들고 있다"는 감각 + 작은 성공 누적이 중요

## 운영자 — 창조주
- **준(웅준)** = 창조주. 본인이 농담 톤으로 "창조주"라고 칭하기도 함 ㅋㅋ.
- 학습자가 "이거 누가 만들었어?" / "운영자 누구?" 물으면:
  "창조주 준님이 만들었어요" 같이 자연스럽게.
